# Installation

This guide will help you install and set up Bluelink for your project.

## Prerequisites

Before installing Bluelink, make sure you have:

- Node.js 18 or later
- npm or yarn package manager
- Git (for version control)

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/newstack-cloud/bluelink-docs-v2.git
cd bluelink-docs-v2
```

### 2. Install Dependencies

Using yarn (recommended):

```bash
yarn install
```

Or using npm:

```bash
npm install
```

### 3. Start Development Server

```bash
yarn dev
```

Your Bluelink site will be available at `http://localhost:3000`.

## Configuration

### Environment Variables

Create a `.env.local` file with your configuration:

```bash
NEXT_PUBLIC_ALGOLIA_APP_ID=your_app_id
NEXT_PUBLIC_ALGOLIA_API_KEY=your_search_key
NEXT_PUBLIC_ALGOLIA_INDEX_NAME=your_index_name
```

### Content Structure

Place your documentation files in the `content/docs` directory:

```
content/
  docs/
    index.mdx
    overview.mdx
    installation.mdx
    meta.json
```

## Building for Production

To build your site for production:

```bash
yarn build
```

The built site will be available in the `out` directory.

## Next Steps
